# Access Denied – File Permissions

## Problem
Cannot open file or folder – "Access Denied" error.

## Solution
1. Right-click file/folder > Properties
2. Click "Security" tab
3. Click "Edit" > select your username
4. Check "Full Control" box under "Allow"
5. Click "Apply" then "OK"
6. If you don't see your username, click "Everyone" and repeat step 4

## If File Is in Program Files
Do not modify system files. Contact IT Help Desk instead.

## Prevention
Do not move files between user accounts without permission check.
