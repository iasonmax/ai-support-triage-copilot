# Second Monitor Not Detected

## Problem
Extended monitor plugged in but not showing up.

## Solution
1. Right-click desktop > Display settings
2. Scroll down to "Multiple displays" section
3. Click "Detect" button
4. Check HDMI/DisplayPort cable is firmly connected to GPU (not motherboard)
5. Try different cable or monitor port
6. If cable is correct, update graphics drivers

## Update Graphics Drivers
1. Right-click Start menu > Device Manager
2. Expand "Display adapters"
3. Right-click your GPU > Update driver
4. Choose "Search automatically for updated driver software"
5. Restart computer

## Prevention
Use high-quality cables. Do not bend cables sharply.
