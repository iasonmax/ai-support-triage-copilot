# USB Drive Not Recognized

## Problem
USB stick plugged in but doesn't appear in File Explorer.

## Solution
1. Open Device Manager (search in Windows)
2. Look for "Unknown Device" under "Other devices"
3. Right-click it > Update driver
4. Choose "Browse my computer for driver software"
5. Click "Let me pick from a list of available drivers"
6. If USB still not recognized, try different USB port
7. If different port works, original port may be broken

## Alternative Fix
1. Right-click USB in Device Manager
2. Click "Uninstall device"
3. Unplug USB, wait 30 seconds
4. Plug back in

## Prevention
Eject USB safely before removing (right-click > Eject).
