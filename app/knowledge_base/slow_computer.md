# Computer Running Slow

## Problem
Computer takes forever to do anything.

## Solution
1. Open Task Manager (Ctrl+Shift+Esc)
2. Click "Performance" tab
3. Look for processes using >90% CPU or RAM
4. Note the process name
5. Go to "Processes" tab, find it, click "End Task"
6. If problem persists, restart computer

## Common Culprits
- Browser with too many tabs
- Antivirus scanning
- Windows Update running
- Malware (run antivirus scan)

## Prevention
Restart computer weekly. Keep only necessary programs running.
