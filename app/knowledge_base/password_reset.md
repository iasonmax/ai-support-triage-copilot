# Password Reset – Active Directory

## Problem
User forgot Windows password and cannot login.

## Solution
1. At login screen, click "I forgot my password"
2. Answer security questions
3. Create new password (minimum 12 characters, 1 uppercase, 1 number, 1 special char)
4. Click "Next"
5. Login with new password

## If Security Questions Don't Work
Contact IT Help Desk with employee ID and two forms of ID.

## Prevention
Write passwords in secure password manager (not sticky notes).
